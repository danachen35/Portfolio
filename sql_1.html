When Capacity Runs Thin: Public Health Insights from 1M Hospital Records

SQL | Database | Excel | Tableau | Storytelling

August 1, 2024 Dana Chen

During my MSBA program, I led an in-depth SQL project to uncover the story behind nearly 1 million COVID-19 hospital capacity records across the United States. What started as a simple query task turned into an eye-opening experience in data cleaning, enrichment, and storytelling. This project was more than SQL syntax ‚Äî it was about making messy public data usable, translating trends into insights, and understanding how real-world systems, like hospital infrastructure, respond in crisis.


From Chaos to Clarity: Data Cleaning & Preparation

The original dataset contained over 964,000 weekly records of hospital activity between 2020 and 2024. However, the path to insight wasn‚Äôt straightforward.
Challenges included:
  • Messy Fields: Inconsistent column names, duplicate hospital IDs, and negative values in numeric fields like bed counts.
  • Long Column Names: MySQL Workbench couldn‚Äôt handle them properly, so manual edits were done in Excel before import.
  • Large File Size: The .csv was nearly a million rows, causing import lags and slowness in MySQL.
Key Cleaning Steps:
  • Used IF() and CAST() functions in SQL to clean up negative values in numeric columns.
  • Converted date strings using STR_TO_DATE() for proper temporal analysis.
  • Removed duplicates and standardized hospital names, city/state labels.
  • Joined with external data (e.g., U.S. Census regions and population estimates) to enhance context and allow normalized analysis (e.g., beds per capita).


Regional Burden: Where Hospitalization Hit Hardest

Once the dataset was cleaned and enriched, we explored how COVID-19 hospitalization trends varied over time and by geography.
Between August 2020 and April 2024, the South and Midwest regions accounted for over 61% of all reported COVID-19 hospitalizations in the U.S.

  • üè• In the Midwest, Illinois and Ohio dominated, accounting for over one-third of regional hospitalizations.
Using SQL aggregations and time-based groupings, we calculated monthly and yearly totals by state and region. This helped surface hidden hotspots and enabled a data-backed assessment of healthcare strain.


Hospital Infrastructure: Comparing Capacity Across States

To assess whether the observed hospitalization patterns aligned with infrastructure readiness, we turned our focus to hospital-related resources at the state level.
We used SQL to summarize and aggregate capacity-related metrics, and visualized the results in Tableau through a series of state-by-state comparisons:
  • üè• Number of hospitals per state: a basic measure of access points across geographies
  • üõèÔ∏è Total inpatient bed capacity: indicative of each state's total surge capability
  • üò∑ Dedicated COVID-19 bed capacity: how states prioritized space for COVID-specific care
These visuals show clear differences in infrastructure scale:
Texas led the nation in both number of hospitals and total COVID-designated beds. California had the highest inpatient bed capacity overall, reflecting its large population base. Florida and Ohio followed closely in both metrics, reflecting substantial‚Äîbut not necessarily sufficient‚Äîcapacity.
However, having more beds or hospitals didn't automatically equate to better outcomes. For example, despite Texas having the most infrastructure in absolute terms, it also experienced some of the highest hospitalization surges, placing significant strain on its system. This state-level infrastructure analysis underscored that preparedness is not just about volume, but about allocation, efficiency, and adaptability in moments of crisis.
